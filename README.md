# Terminal Emulator (WIP)

A minimal GPU-accelerated terminal emulator written in C++ with a custom rendering pipeline and PTY integration.
It's for personal use, not really an open projects, so if you decide to use it, expect bugs and problems.

## Build

```bash
mkdir build
cd build
cmake ..
make
./Terminal
```
